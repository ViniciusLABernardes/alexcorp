AlexCorp

Desenvolvido por:

Vinícius Leandro de Araujo Bernardes


AlexCorp é um projeto pessoal visando meu desenvolvimento e testando minhas habilidades tanto em front-end, como em back-end(*incluindo o banco de dados).



Recursos Principais:


Comentários interativos para compartilhar opiniões e insights.


Interface responsiva para uma experiência de usuário intuitiva.


Backend construído em PHP e MySQL para gerenciar dados do usuário.


Utilização de PHPMyAdmin para facilitar a administração do banco de dados MySQL.


Utilização de JSON para armazenar e fornecer dados adicionais.



Tecnologias Utilizadas:


front-end:HTML, CSS, javascript.


back-end:PHP.


banco de dados: MySQL
